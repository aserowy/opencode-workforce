# User Story: Install tools directory
- ID: US-0007
- Status: done
- Feature: FEAT-0003
- As a: workflow/agent maintainer
- I want: tools/ to be included by install.sh
- So that: required tools are available after installation without manual steps
- Acceptance criteria:
  - Given install.sh is run in a fresh environment
    - When installation completes successfully
    - Then the tools/ directory is installed and available for use
    - And tools/ are installed into ~/.config/opencode/
  - Given install.sh is run multiple times
    - When it re-installs the project
    - Then including tools/ does not require manual intervention
